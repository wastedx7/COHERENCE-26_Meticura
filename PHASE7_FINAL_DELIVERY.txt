📊 COMPONENT ARCHITECTURE

  Component Hierarchy:
  
  BudgetAnalysisDashboard (Container)
  ├─ Summary Cards (3 KPI cards)
  ├─ LapseRiskPanel (Interactive)
  │  └─ Critical & High risk lists
  ├─ AnomalyInsightsPanel (Display)
  │  └─ Severity distribution
  ├─ BudgetStatusGrid (Display)
  │  └─ Top 6 departments
  └─ BudgetComparisonChart (Visualization)
     └─ Recharts BarChart

🔌 API INTEGRATION

  Endpoints Used: 3
  
  1. GET /api/lapse/
     └─ Used by: usePredictions hook
     └─ Returns: 500 lapse predictions
     └─ Features: risk_level, days_until_lapse, r2_score
  
  2. GET /api/anomalies/
     └─ Used by: useAnomalies hook
     └─ Returns: Active anomalies list
     └─ Features: Severity, violations, combined verdict
  
  3. GET /api/budget/
     └─ Used by: useBudget hook
     └─ Returns: Department budget allocations
     └─ Features: allocated, spent, utilization%

═══════════════════════════════════════════════════════════════════════════════
                         ERROR HANDLING
═══════════════════════════════════════════════════════════════════════════════

🛟 ERROR SCENARIOS HANDLED

  API Failure
  └─ Predictions error        → ErrorState with retry
  └─ Anomalies error          → Empty list + message
  └─ Budget error             → ErrorState with retry

  Data Issues
  └─ Missing predictions      → Loading spinner
  └─ No anomalies            → "No anomalies detected" message
  └─ No budgets              → "No budget data" message

  Network Issues
  └─ Timeout                 → Retry mechanism (3 attempts)
  └─ 500 error               → Error message + retry button
  └─ Connection lost         → Retry on reconnect
